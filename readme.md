# Expense dashboard

You can use this application for tracking your expences & see live data through it dashboard. You can navigate through pages easily, using links you can see at the left of your screen in this app. It's my second project where i built crud system with separate dashboard.

# Features
- Add Items category wise,
- Persistent Storage using Browser APIs,
- Look through Items category wise through it's Dashboard
- Clean & Calm UI,
- 
# Technologies used
- HTML,
- CSS,
- Vanilla JavaScript
- Browser APIS(DOM, LocalStorage. etc)

# How It Works
- Items are stored locally in the browser, using it APIs
- JavaScript handles CRUD operations & Other Logic,
- UI updates dynamically without page reload,
- Implimented full Single Page App logic using vanilla js,

# Usage
- Clone the repository,
- Open 'Index.html' in a browser,
